Objective System's Android Sample Project
------------------------------------------

Prerequisites:
1) Android Studio is installed (https://developer.android.com/studio).

In order to use this sample, you should do the following:

1) Open a command prompt in this folder (AndroidSample)
2) Run 'ac.bat' to compile the schema and generate code.
3) Start Android Studio.  Open an existing Project by browising to this folder.
   Android Studio will create the local.properties file in this directory
   automatically.
4) Make sure you have an emulator device set up.
5) Run the application in Android Studio to build and install the sample to
   your emulated device.
