Example of an H.323 user information message.  This sample demonstrates 
the use of Print to Stream Functionality. It shows how to define a simple
callback function to direct print streams to a file. The reader.cpp shows
how to register the callback as a global callback, where as writer.cpp 
shows how to register a context level callback.

More detailed H.323 sample and test programs can be found in the Objective 
Systems H.323 framework.  Contact info@obj-sys.com for information on 
obtaining this software.
